# SkillHot 视频视觉规范

## 网站摘要

- 产品：SkillHot，GitHub Agent Skills 与配套工具的中文发现、分类和榜单网站。
- 主色：墨黑 `#111114`、白色 `#FFFFFF`、浅灰 `#F7F7F8`、边框灰 `#E4E5E8`、强调珊瑚红 `#F2555F`。
- 视觉语言：轻盈、现代、编辑感强；细边框、克制圆角、黑白插画、少量珊瑚红提示，不使用渐变。
- 实际素材：网站滚动截图、搜索与详情演示、榜单与分类页面、GitHub 项目预览。

## 画布与排版

- 画布：1920 × 1080，30fps，横屏。
- 安全区：左右 96px，上 76px，下 92px；字幕基线位于底部 72–116px。
- 字体：中文使用 `PingFang SC` / `Hiragino Sans GB`，英文与数字使用 `Arial`，技术标签使用 `SFMono-Regular`。
- 标题：72–112px，900 字重，最多两行；辅助文字 28–36px。
- 字幕：44px，750 字重，黑底白字或白底黑字，单组不超过两行。

## 核心构图

- 每个场景只保留一个主视觉焦点。
- 实际网站媒体占画面 76%–88%，使用带细边框与浅阴影的浏览器窗口容器。
- 说明卡片最多 3 个，单卡只表达一个指标或功能。
- 画面背景为暖白 `#FAF9F6`，深色强调场景使用 `#111114`。

## 动效

- 入场：主窗口从 `y: 28`、`scale: .97`、`opacity: 0` 进入，0.65–0.9s。
- 文案：逐组上移 20px 淡入，间隔 0.08–0.12s。
- 强调：搜索框、分类标签和数据指标用珊瑚红描边/底色短暂点亮。
- 转场：推入为主，模糊交叉淡化为辅，唯一一次轻微 zoom-through 用于从搜索到详情。
- 不做连续漂浮、弹跳和炫技粒子；遵循确定性 `fromTo` 时间线。

## 品牌尾板

- 左侧 Sav同学圆形头像与签名，右侧写 `SkillHot · Agent Skills 中文导航`。
- CTA：`入口看主页`、`关注 Sav同学`。视频画面和字幕不出现完整访问链接或完整域名，避免平台识别为外链导流。
- 保持白底、黑字、珊瑚红按钮，延续网站气质。

## 对外表达边界

- 站在第一次看到网站的用户视角介绍：它是什么、能解决什么问题、怎么找、详情页看什么、为什么值得收藏。
- 不展示内部修复记录、分类纠偏过程、成本指标、开发任务清单或“我让你修改”的表达。
- 不把单个项目当成纠错案例讲解；示例只作为搜索/详情/榜单的自然演示素材。
