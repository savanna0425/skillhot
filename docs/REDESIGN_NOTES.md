# SkillHot Web 重构说明

这份文档记录面向项目维护者的产品决策，不在网站公众页面展示。

## 布局

- 删除固定高度、居中圆角外壳和灰色“桌面背景”。
- 页面使用浏览器原生纵向滚动；Header 固定，侧栏各自 Sticky。
- 左侧分类栏和右侧详情栏均可收起。
- 移动端左栏变为筛选抽屉，右栏变为详情底部面板。

## 页面定位

- 发现：编辑式探索，由分类入口、本周新发现、精选合集和随机漫游组成。
- 榜单：结构化排名表，支持完整分类和三种排序。
- 分类：展示统一分类体系及选中分类结果。
- 话题：以生态规模和活跃度重新组织 GitHub Topics，不显示原始搜索分页。

## 视觉状态

筛选按钮、分类卡片、话题卡片和被选中的 Skill 使用统一规则：未选中为白色，选中为黑色。黑色不再被用作随机装饰卡片。

## 公共文案

网站只保留面向所有访问者的产品说明、数据来源和贡献入口。部署取舍、更新成本与需求响应记录放在项目文档中。
